# EventHub — WordPress тема для міських подій

Кастомна WordPress тема для порталу міських івентів, розроблена "з нуля". Проєкт демонструє навички створення динамічних шаблонів, роботи з Custom Post Types та оптимізації сайту для реального хостингу.

**🔗 Live Demo:** [[https://dev-city-events.pantheonsite.io](https://dev-city-events.pantheonsite.io)](https://dev-city-events.pantheonsite.io/)

## 🛠️ Що реалізовано:
- **Custom Post Types:** Створено окремий тип записів `event` для керування афішею.
- **ACF (Advanced Custom Fields):** Налаштовано кастомні поля для виводу дати, часу, локації та зображень подій.
- **Динамічні цикли (WP_Query):** 
  - Вивід "Найближчих подій" на головній сторінці.
  - Блок "Схожі події" на сторінці івенту з виключенням поточного поста.
  - Сортування за мета-полем дати (`card_date`) від найближчих.
- **Vanilla JS:** Плавний скрол до форми, "липкий" хедер та адаптивне мобільне меню.
- **Оптимізація:** Налаштовано кешування, мініфікацію ресурсів та Lazy-load для зображень.

## 📦 Стек технологій та плагінів:
- **Core:** PHP (WordPress Templates), HTML5, CSS3 (BEM), JavaScript (ES6).
- **Контент:** **ACF** (поля та реєстрація CPT), **Font Awesome** (іконки).
- **SEO & URL:** **Yoast SEO** (мета-теги), **Cyr-To-Lat** (людинозрозумілі посилання).
- **Продуктивність:** **Autoptimize** (оптимізація CSS/JS).
- **Зворотний зв'язок:** **Contact Form 7**.
- **Деплой:** **All-in-One WP Migration** (перенесення з локального середовища на Pantheon).
